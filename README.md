# Проект: Mesto

- [Ссылка на проект](https://kejjero.github.io/mesto/)

**Описание:**
Сайт является аналогом instagram, где вы можете загружать свои фотографии и отмечать <3
**Используемые технологии:**

- HTML/CSS
- Grid
- Flexbox
- media-запросы
- Адаптивная кроссбраузерная верстка
- BEM
- JavaScript
